# TENGU v1.0
Tengu is a Phishing tool based on BlackEye by @thelinuxchoice. It comes with a default template and ability to import custom pages.

## How to use:

```
git clone https://github.com/Smarttfoxx/tengu/
cd tengu
bash tengu.sh
```
